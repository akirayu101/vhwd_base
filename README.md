vhwd_base
=========

vhwd_base is a lightweight, cross-platform C++ library for  general purpose.
It supports linux(32bit version is not tested) and windows platform currently,
including following contents:

* **basic**
* **collection**
* **logging**
* **threading**
* **net**
* **ipc**
* **memory**
* **serialization**
* **scripting**
* **xml**
